A pult IP címének beállítása

Egy IP cím egy megadott tartományban szükséges az eszközök összekapcsolásához. Ez az oldal a statikus IP cím beállításáról szól a pulton. Ha az onPc-n akarod beállítani az IP címet akkor kövesd ezt a linket.

Ha olyan hálózatod van ami DHCP-t használ, akkor engedélyezheted a DHCP szervernek az IP címek kiosztását. Ezekről bővebben a DHCP használata az MA eszközökben fejezetben olvashatsz.

Három módja van az IP cím beállításának a pulton. Mindegy melyiket választod, mindenképpen újra kell indítani a pultot, hogy a változások érvénybe lépjenek..

Az IP cím beállítása parancssorból

Használd a SetIP parancsot, hogy megnézd és módosítsd az IP címet a pulton. Ez a parancs használható az Ethernet csatlakozó váltásához, és átjáró beállításához is. Kövesd a linket, hogy olvashass a parancsról.

IP cím beállítása az MA hálózati beállításaival

Megváltoztathatod az IP címet a pulton, ha megnyomod a Setup és utána az MA Network Control gombot.

Itt csak az 1. számú Ethernet vezérlő IP címét lehet megváltoztatni, amikor a pult nincs munkamenethez csatlakoztatva. A menü fejlécében láthatod az aktuális állapotát a pultnak. Ha más van kiírva mint "Standalone", akkor bökj rá a Leave Session gombra.

Ha pult standalone módban van, akkor bökj a zöld beviteli mezőre a "Station IP" mellett. Utána írd be az új IP címet - és indítsd újra a rendszert.

IP cím beállítása az MA Network Configuration segítségével

Megváltoztathatod a pult IP címét ha megnyomod a Setup és utána a MA Network Configuration-ba lépsz.

Ebben a menüben válaszd ki a Consoles fület. Itt a rendszerben lévő összes pult látható. Kiválasztva egy oszlopot ami IP címmel rendelkezik a táblázatban, majd a képernyő görgetőre nyomva, kapsz egy ablakot, ahol megváltoztathatod az IP címet. Minden pultnak két IP címe van - minden egyes Ethernet csatolónak egy.

A második IP címet meghagyhatod mindig az Art-Net címnek. Az Art-Net címek mindig két határérték között lehetnek - az egyik 2.-vel kezdődik és a másik 10-el. Más gyártók gyakran használják a "10"-es tartományt az elsődleges címzéshez. Emiatt is hasznos gyakorlat, ha az Art-Net-et a "2"-es tartományban tartod. Ez megelőzi a gondokat, azokon a hálózatokon, ahol nem csak MA eszközök csatlakoznak és nem állíthatod őket.

Van egy oszlop a DHCP bekapcsolásához. Olvass a DHCP-rőlRead a DHCP használata az MA eszközökben fejezetben.
