Effect Gomb - Effekt Gomb

Az Effect gomb, jobbra van a View gombtól.

 

  • Megnyomva az Effect gombot, beíródik az Effect kulcsszó a parancssorba.
[Channel]> Effect

 

  • Ha egy funkciós kulcsszó, mint a Store már szerepel a parancssorban, megnyomva az Effect Effect gombot, beíródik a Form kulcsszó a parancssorba.
[Channel]>Form

 

  • Nyomvatartva az Effect gombot, 2 másodpercig megnyílik a futó effektek ablakocska.
     
     
  • Megnyomva az  + Effect végrehajtódik a SyncEffects parancs.
     
  • Nyomvatartva az List + Effect gombot, megnyílik az ideiglenes effekt tároló.
