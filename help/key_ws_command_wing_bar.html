Command wing sáv

A command wing sávval tudod vezérelni az 1-6-ig a húzókat, és a 101-106-ig a gombokat.

MA onPC command wing és külsõ képernyõ

Ha az MA onPC command wing külsõ képernyõvel használod, nyisd meg a command wing sáv ablakot.

  • Bökj egy üers területre, válaszd a Other fület és bökj a Command Wing Bar-ra.

A command wing sáv ablak megnyílt.

Command wing grandMA2 onPC

Ha a grandMA2 onPC-t, MA onPC command wing-el használod, akkor kapcsold be a command wing sávot.

A command wing sáv bekapcsolása a 2. képernyõn:

  1. Kattints a sárga golyóra  az ablak bal felsõ sarkában az grandMA2 on PC-ben.
  2. Kattints az Options....
  3. Kattints a Viewfülre és alatta válaszd ki a CmdWing Bar On-t.

A command wing sáv most látható a 2. képernyõ alján, balra az encoder sávtól a grandMA2 onPC-ben.
