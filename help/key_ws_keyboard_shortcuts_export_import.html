Billentyûparancsok exportálása vagy importálása

Fontos:
A billentyûparancsok csak a grandMA2 onPC-n érhetõk el.

A testreszabott billentyûparancsokat késõbbi importálás céljából exportálhatod egy másik show-fájlba.

Billentyûparancsok exportálása

  1. Kattints a Setup elemre, majd a console fül alatt, kattints a Edit Keyboard Shortcuts elemre.
    Megnyílik az Edit Keyboard Shortcuts(Billentyûparancsok szerkesztése) ablak.
  2. Kattints az Export elemre.
    Megnyílik az Export Keyboard Shortcuts(Billentyûparancsok exportálása) elõugró ablak.
  3. Válaszd ki az exportált billentyûparancsok meghajtóját.
  4. Ha felül akarsz írni egy meglévõ .xml fájlt, kattints a felülírni kívánt fájlra. Ha új fájlt szeretnél létrehozni, írd be az új fájl nevét a zöld szerkesztési sorba .
  5. Kattints az Export gombra.

A billentyûparancsok .xml fájlt exportálódtak.

Ha a billentyûparancsok .xml fájlját a számítógép meghajtójára exportáltad, akkor az export fájl az C:\ProgramData\MA Lighting Technologies\grandma\gma2_V_3.1.2\shortcuts rejtett mappájában található. >.

Ha a billentyûparancsok .xml fájlt exportáltad egy USB-meghajtóra, akkor az exportálási fájl az E:\gma2\shortcuts között található.

Gyorsbillentyûk importálása

  1. Kattints a Setup elemre, majd a console fül alatt, bökj az Billentyûparancsok szerkesztése elemre.
    Megnyílik a Edit Keyboard Shortcuts(Billentyûparancsok szerkesztése) ablakra.
  2. Kattints az Import gombra.
    Megnyílik a Billentyûparancsok importálása elõugró ablak.
  3. Válaszd ki azt a meghajtót, amelybõl a billentyûparancsokat importálni szeretnéd.
  4. Kattints a fájlra, majd kattints az Import gombra.

A kiválasztott billentyûparancsok importálva lettek.
