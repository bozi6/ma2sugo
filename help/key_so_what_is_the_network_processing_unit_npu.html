Mi az a Network Processing Unit (NPU)

Az MA Network Processing Unit (NPU) MA Hálózati Feldolgozó Egység paraméter kiterjesztő kütyü Ami további DMX portokat ad hálózati megoldással.

Ez egy 19"-es 3 egység magas rackbe szerelhető processzor.

Van 8 DMX csatlakozója és 1 etherCON csatlakozója a hálózathoz.

Önálló egységként nem üzemelhet. Csatlakoznia kell egy sessionhoz, hogy bármit is lehessen vele kezdeni.
