Ideiglenes szűrők használata

A szűrő tárolóban lévő és előre beállított szűrőkön kívül, használhatsz ideiglenes szűrőket is.

Ugyanazokat az akciókat tudod végrehajtani velük mint a filter tárolóban lévőkkel.

A következő szöveg leírja, hogyan tudsz megnyitni ideiglenes szűrő ablakocskát. De van még a Parancs Szűrő ablakod is valamelyik képernyőn, ezt is használhatod.

A következő szövegben, felfedezzük az ideiglenes szűrőket néhány példán keresztül, ugyanolyan példákon, mint amit a Szűrők használata programozáskor fejezetben. Ha még nem olvastad volna, akkor itt a remek alkalom.

Temporary Store Filter - Ideiglenes tároló szűrő(ITS)

Elérheted az ITS-t a Store gomb nyomvatartásával. Most látod a Store Encoder Toolbar-t a 2. képernyő alján. Ebben az eszköztárban van egy gomb amit Store Filter-Tároló Szűrőnek hívnak. Rábökve megnyílik egy szűrő szerkesztő ablakocska. Ez ugyanúgy működik mint ahogy a Szűrők készítése fejezetben láttuk. Van két extra gomb a jobb oldalon, ezek a "Reset Filter on Clear-Szűrő törlése törléskor" és "Store Filter and Select-Szűrő tárolása és kiválasztása".

Reset Filter on Clear:

Általában a szűrőt visszaállítjuk amikor végrehajtod a (tárolás) akciót, de lehet, hogy kellene még a szűrő. Akkor aktiváld ezt a gombot. Ilyenkor csak a Clear(All) parancs törli a kiválasztott szűrőt.

Store Filter and Select:

Erre a gombra bökve az aktuális szűrőbeállítások elmentődnek a következő szabad szűrő tároló helyre, és megkérdezi a nevét. Az új szűrő is kiválasztódik ilyenkor, nem csak az eddig alkalmazott szűrő.

Példa az ideiglenes tárolószűrőre

A következő példa ugyanaz mint amit a Szűrők használata programozáskor fejezetben mutattunk.

Megnézünk egy praktikus helyzetet, amikor a szűrők segítségével gyorsabban lehet dolgozni.

  1. Használd a "Demo Dimmer and more" demo showt vagy egy showt néhány lámpával.
  2. Nézd meg, hogy a kiválasztott szűrő az "All" legyen, tehát semmi ne legyen szűrve.
  3. Válassz ki néhány lámpát és adj nekik értékeket, a dimmert is beleértve.

Képzeld el ez az amit szeretnél egy képben látni - ez legyen pl. a 2. kép.

Nem akarunk MIB-et használni, de azt akarjuk, hogy a lámpa már álljon be a helyére amikor a 2. képet adjuk. Tehát minden adatot elmentünk, kivéve a dimmer értékét az 1. képben és a dimmer adatát a 2. képben.

  1. Nyomd meg a Store és utána a Store Filter gombokat a kódoló sávban.
  2. Kapcsold ki a "Dim" attribútumot a Tároló szűrő ablakocskában.
  3. Nyomj meg egy executor gombot - létrehozva az 1. képet egy új szekvenciában és executorban.

Most mindent elmentettél kivéve a dimmer értékeket az 1. képben. A dimmer értékek még aktívak a programozóban, a maradék eltárolódik. Az ideiglenes szűrő most eltűnik és semmi nem szűrődik továbbá.

  1. Nyomd meg a Store és ugyanazt az executor gombot.
  2. Az ablakocskában válaszd a Create Second Cue-t.

Ez lett a második kép. Töröld a programozót és próbáld meg futattni a képet.

Temporary At Filter - Ideiglenes At szűrő(TAF)

A TASF olyan mintha a szűrő tárolóban lennél.

Tartsd nyomva az At gombot kb. .1 másodpercig és megnyílik a szűrő ablakocska.

Ebben az ablakban van két extra gomb, ami már fent le lett írva.

Alul találsz még három extra gombot. Ezek színtranszformáló gombok. Arra használjuk, hogy mi történjen ha a forrás és a cél lámpának nincs azonos szín attribútúma.

MixColor:

Ez a MixColor rendszert használja. A beállításoktól függően lehet Open értéket állítani bármely színkörben.

Color Wheel:

Ez a legközelebbi színt használja a színkeréken. Beállítástól függően ez is beállítható Open értékre a MixColor rendszerben.

MixColor + Col Wheel:

Ez mindkét lehetőséget használja a legjobb eredmény elérése érdekében.

Példa

A következő példa, ugyanaz amit már a Szűrők használata a programozáshoz fejezetben használtunk.

Ebben a példában csak a színeket vesszük ki az egyik lámpából és átteszük a másikra.

  1. Használd a "Demo Dimmer and more" demo showt, vagy egy másik showt amiben néhány lámpa van és néhány szűrő.
  2. Legyen a kiválasztott szűrő az "All" szűrő.
  3. Válassz lámpát és adj nekik értéket néhány attribútúmára a dimmer, pozíció, és mixcolor értékeket is, hogy új színben usszon át.
  4. Válassz egy másik lámpát és adj neki egy másik pozíciót és dimmer értéket.

Most csak a szín értékeket vesszük az első lámpából. Nem akarjuk a pozíció és dimmer értékekt elvenni, csak a színeket.

  1. Tartsd nyomva az At gombot ameddig a szűrő ablakocska megjelenik.
  2. Bökj a None gombra a fejlécen, tehát ne legyen attribútum kiválasztva.
  3. Tap the Color preset type so it turns on. This will allow all color information to pass the filter.
  4. Tap the first fixture on the screens (or use the keys - Fixture [number] Please).

Now both fixtures have the same color values.

Temporary filtering using the If key

While storing or using the At command you can include an encapsulated condition. The condition can be almost anything and can include many different elements.

The condition is encapsulated using the If key. You can keep the key pressed if you are making a complex condition or press it shortly if it is a more simple condition.

The encapsulation is between the If command and the EndIf command. EndIf is available as a command using the If key after you have typed the first If command. Sometimes the software will automatically add the EndIf command.

The condition is usually made by pressing different elements on the screens, but it can just as well be written commands. The condition can be positive or negative. For example: If you want to only have position values then you can simply tap the Position button in the Preset Control Bar. If you want everything except the position values, then you can press the - key and then the Position button to get the following command: If - PresetType "POSITION" EndIf in the command line.

 

Examples

The two examples used above can also be achieved using the If key.

  1. Use the "Demo Dimmer and more" demo show or a show with some fixtures.
  2. Make sure the Selected filter is the "All" filter so nothing is filtered already.
  3. Select a fixture and give it values on several attributes including Dimmer.

So now we imagine that this is the look we want in a cue - it is going to be cue number 2.

We do not want to use the move in black function but we want the fixture to already be pre-positioned when we run cue number 2. So we are going to store all data except the dimmer in cue number 1 and then the dimmer value in cue number 2.

  1. Press Store.
  2. Press If and then -. Now the filtering is active and we have already selected that we want to exclude something.
  3. We want to exclude the dimmer so now you can tap the Dimmer button in the Preset Control Bar.
  4. Finish the temporary filter by pressing the If key again. Now the command line should read: Store If - PresetType "DIMMER" EndIf.
  5. Press an empty executor button - creating a new cue 1 in a new sequence and executor.

Now you have stored all values except the dimmer in cue number 1. You still have the dimmer values active in your programmer, the rest is stored. The temporary filter is now gone and noting is filtered anymore.

  1. Press Store and then the same executor button.
  2. In the pop-up select Create Second Cue.

This was an example on  how to use the If command to exclude an attribute - actually an entire preset type.

 

The next example is about allowing some attributes to pass the filter.

In this example we are going to grab only the color values from one fixture and apply it to another.

  1. Use the "Demo Dimmer and more" demo show or a show with some fixtures and some filters.
  2. Make sure the Selected filter is the "All" filter.
  3. Select fixture 1 and give it values on several attributes including Dimmer, Position, and use the MixColor attributes to blend a new color.
  4. Select a different fixture and give it another position and dimmer values.

Now we would like to grab the color values only from the first fixture. We do not want to grab the position and dimmer values from the fixture, only the color.

  1. Press At Fixture 1 If. Now you can select the filter you want to use.
  2. We need all the color information so tap the Color button in the Preset Control Bar. The "EndIf" command is automatically added when a single thing is tapped after pressing the If key
  3. Press the Please keys to execute the command.

Now both fixtures have the same color values.
