Az MA Gomb

Az gomb a parancs rész alatt van a 0 gomb alatt.

 

Nyomd meg az gombot, hogy az összes aktív ikonját az üzenetközpontnak megjelenítse a parancssorban. Ez a parancs megmutatja azokat az ikonokat is, aminek az állapota No (= üres)-re van állítva, az üzenetközpont beállításaiban. Bővebben az üzenetközpont ikonjairól. Bővebben az üzenetközpontról: Üzenet központ.

A következő táblázat tartalmazza a gyorsbillentyûket amik az gombbal mûködnek.

Nyomd meg ezt a gombot Parancs Leírás
+ szám Nincs parancs Kiválasztja a preset típusát a preset vezérlő sávban amit a preset típus száma jelez.
+ szám + ugyanaz a szám mégegyszer Nincs parancs Végigvált a képesség típusokon az adott preset típus sávban.
+ . Default Beírja a Default kulcsszót a parancssorba.
+ At Stomp Beírja a Stomp kulcsszót a parancssorba.
+ At + At Extract Beírja az Extract kulcsszót a parancssorba.
+ Down Nincs parancs Lejjebb hajtja a beépített monitorokat.
+ Effect SyncEffects Végrehajtja a SyncEffects kulcsszót.
 + Fix Nincs parancs Ki/be kapcsolja a Pult lezárását
+ Fixture FixtureType Beírja a FixtureType kulcsszót a parancssorba.
+ Full FullHighlight Végrehajtja a FullHighlight kulcsszót a parancssorban.
+ Goto LoadNext Beírja a LoadNext kulcsszót a parancssorba.
+ Goto + Goto LoadPrev Beírja a LoadPrev kulcsszót a parancssorba.
+ Group Layout Beírja a Layout kulcsszót a parancssorba.
+ Group + Group SearchResult Beírja a SearchResult kulcsszót a parancssorba.
+ Group + Group + Group Mask Beírja a Mask kulcsszót a parancssorba.
+ List List Beírja a List kulcsszót a parancssorba.
+ Macro Timer Beírja a Timer kulcsszót a parancssorba.
+ Macro + Macro Plugin Beírja a Plugin kulcsszót a parancssorba.
+ Move Search Beírja a Search kulcsszót a parancssorba.
+ Next NextRow Végrehajtja a NextRow parancsot.
+ Preset PresetType Beírja a PresetType kulcsszót a parancssorba.
+ Prev PrevRow Végrehajtja a PrevRow parancsot.
+ Set Nincs parancs Feloldja a beépített monitorféket, hogy kézzel lehessen a monitor dőlésszöget beállítani.
+ Store Record Beírja a Record kulcsszót a parancssorba.
+ Time Nincs parancs Megnyitja az effekt réteget a kódoló sávban.
+ Time + Time Nincs parancs A következő effekt rétegre vált a kódoló sávban.
+ Up Nincs parancs Felfelé mozgatja a beépített monitorokat.
 + Update Nincs parancs Listázza a frissítéseket a parancssor visszajelző ablakban. 
+ View Screen Beírja a Screen kulcsszót a parancssorba.
+ V key Nincs parancs Kiválasztja a V gomb számú oldalhoz tartozó oldalt.
