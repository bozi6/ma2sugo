Remote Kulcsszó

A remote kulcsszóhoz, írd be Remote a parancssorba, vagy röviden Rem.

Leírás

A remote kulcsszó egy objektum kulcsszó, amivel elérheted a távoli bemenet típusokat.

Tárolhatod vagy törölheted a távoli bemenet típusokat és paramétereket rendelhetsz hozzájuk.

További információkért, lásd Mik a távoli bemenetek? és Távoli bemenet beállítása fejezeteket.

Szintaxis

[Funkció] Remote [Távoli bemenet ID] . [ID]

Assign Remote [Távoli bemenet típus ID] . [ID] / [Paraméter] = [Paraméter Érték]

A következõ táblázat bemutatja az elérhetõ távoli bemeneteket a távoli bemenet típus ID-vel együtt.

Távoli Bemenet Távoli Bemenet Típus ID
Analog Remote 1
MIDI Remote 2
DMX Remote 3

Amikor távoli bemenet típust tárolsz, akkor az ID-k nek sorrendben kell lennie és 1.-tõl kell kezdõdniük.

Paraméter

A következõ táblázat megmutatja a hozzárendelhetõ paramétereket a parancssoron keresztül, az Assign kulcsszó használatával.

Paraméter Paraméter Érték Leírás
name "remote name" Távoli nevet rendel hozzá.
Csak analóg távoli bemenetnél:
input		 1...64 Hozzárendeli a bemenetet a csatlakozott DC távvezérlésbõl.
Csak MIDI távvezérlésnél:
note		 0...1023 MIDI hangjegy hozzárendelése.
Csak MIDI távvezérlésnél:
channel		 1...16 or "All" MIDI csatorna hozzárendelése.
Csak DMX távvezérlésnél:
DMX		 1.001 to 256.512 [univerzum]. [dmx cím] DMX univerzum és cím hozzárendelése.
type "None", "Exec", "CMD", "Hardkey"
Csak DMX távvezérlésnél:
"Programmer"		 Meghatározza milyen akció történjen a pulton amikor a kapcsolat aktív.
page 0...9999 vagy "current" Exec. típusnál. Hozzárendeli az executor oldal számát. Az aktív oldal az aktuális..
executor 1...220 Exec. típusnál. Az executor számát rendeli hozzá.
button "Button 3", "Button 2", "Button 1", "Fader" Exec. típusnál. Hozzárendeli a gombokat, vagy a húzót amit a pultnak aktiválni kell.
keycode "every hardkey", e.g. "X1", "Go", "List" Hardkey típusnál. Hozzárendeli a gombot amit a pultnak aktiválni kell.
cmd "a command" e.g. "Group 1" CMD típusnál. Hozzárendeli a parancsot amit a pultnak végre kell hajtania.
info "additional information" További információ megadása.

 

Példák

  • Store new DMX remote.
[Channel]> Store Remote 3.1

 

  • Assign an existing remote a DMX address, a remote type and a command.
[Channel]> Assign Remote 3.1 /dmx=5.1 /type=cmd /cmd="Group 1"
