Show mentése új néven

Az aktuális show-t el lehet menteni új néven.

A GUI használatával

Megnyomva a Save Show as-Show mentése mint gombot, megnyílik a "Please enter filename"-Kérem adjon meg egy fájlnevet ablakocska, ahol új nevet lehet adni a show fájlnak.

Megnyomva a gombot akkor megnyílik a képernyő billentyûzet. Amikor rányomsz a Please/Enter-re akkor a show file ezen a néven mentődik el. A mentés is megszakítható az X -re kattintással.

A parancssor használatával

A show fájl elmenthető, új néven, a parancssor használatával is. A parancs SaveShow [uj_nev]. Olvass róla bővebben itt.
