Gyorsbillentyûk hozzáadása és törlése

Fontos:
A gyorsbillentyûk, csak a grandMA2 onPC-n elérhetõek.

További gyorsbillentyûket, vagy meglévõket tudsz hozzáadni/törölni.

Gyorsbillenty hozzáadása

  1. Kattints a Setup és a konzol kattints az Edit Keyboard Shortcuts.
    A gyorsbillentyû szerkesztõ megnyílik.
  2. Válaszd ki hogy a rövidítést hogyan akarod hozzáadni:
    - Direct Hozzáféréssel
    - Ctrl Hozzáféréssel
    - Alt Hozzáféréssel
    - Ctrl Alt Hozzáféréssel
  3. Kattints az Add-ra, a táblázat alatt, ahol hozzáadtad a kombinációt.
    Az Add Shortcut ablakocska megnílik a kiválasztott táblához képest.
  4. Kattints a gombra amihez hozzá akarod adni.
    A Select Virtual Key Code ablakocska elõjön.
  5. Kattints a PCgombra amit szeretnél.
    Az ablakocska bezáródik.

A gyorsbillentyû hozzáadódott.

Gyorsbillentyû törlsée

  1. Kattints a Setup-ar és a console fül alatt kattints a Edit Keyboard Shortcuts.
    A gyorsbillentyû szerkesztõ megnyílik.
  2. Kattints rá a a gyorsbillentyûre amit törölni szeretnél.
  3. Kattints lejjebb a táblázatban a Delete gombra.

A gyorsbillentyû törölve lett.
