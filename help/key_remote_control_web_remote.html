Web Hozzáférés

Az állomás távolról is vezérelhetõ bármilyen böngészõvel ami a rendszerhez csatlakozik.

A web hozzáférés nagyon rugalmas. A képernyõ mérete és felbontása függvényében, megpróbál annyi információt kijelezni, amennyi csak lehetséges. Ez azt jelenti, hogy sok nézet sok információt tud megjeleníteni - különösen nagy felbontású kijelzõn. A képeket amiket itt használunk a leírásban tipikusan a legkisebb lehetséges méretre vonatkoznak.

Engedélyezni kell a Globális Beállításokban: Setup -> Console -> Global Settings -> Remotes

Itt van egy beállítás amit "Remotes"-nak hívnak és két állapota van: Login Disabled-bejelentkezés tiltva és Login Enabled-bejelentkezés engedélyezve.

 

Egy böngészõt megnyitva a számítógépen, vagy kézbevehetõ eszközön és az állomás IP címének beírásával, máris vezérelhetjük azt.

Utána egy bejelentkezõ ablak jön elõ:

Nem lehetséges az alapvetõ "Administrator" felhasználó bejeentkeztetése.

Maximum három távoli hozzáférés engedélyezett egyszerre.

Valós felhasználó bejelentkezése után a web felület elérhetõ.

Fixture sheet - lámpa lap

A lámpa oldal információkat nyújt a lámpákról. Részletek aLámpa oldalról ezen a linken találhatóak Lámpa támblázat fejezet.

Van néhány külön gomb az ablak tetején a Telefonos módban és az ablak alján tablet módban. A következõkben leírjuk ezeket:

  • Clear:
    Ez mint a normál Clear-törlés gomb.
  • Highlight:
    Ez ki/bekapcsolja a Highlight funkciót - mnit normál esetben Highlight gomb.
  • Fixture Sort:
    Ez a Fixture Sort-lámpa sorbarendezés funkció. Az alábbi linken vannak infók lejebb.
  • Feature Sort:
    Ez a Feature Sort-képesség sorbarendezés funkció. Az alábbi linken vannak infók lejebb.
  • Prg Only:
    Ez bekapcsolja a Programmer Only-csak a programozó maszkot.
  • Preset Value selector:
    Rábökve a csúszkagombra választhatsz az alábbiakból: Value, Fade, Delay, és Output. Csúsztatva, megnyílik egy ablakocska amivel kiválaszthatod a réteget közvetlenül. A rétegekrõl bõvebben a Rétegek az oldalakon fejezetbenolvashatsz.

    Ez megváltoztatja a következõ executor készletet az aktuális ablakon.

    Ez átvált az elõzõ executor oldalra.
  • Next page:
    Ez átvált a következõ executor oldalra.

    Ez az utolsó gombhoz van kapcsolva (Button) - csak egy lehet kiválasztva. Erre bökve megjelennek a Fader Executors.

    Ez is kapcsolódik a efnti gombhoz - csak egy lehet kiválasztva. Erre bökve megjelenek a Button Executors.
