Modul Kezelő

Ha ehhez a fejezethez egy linkről érkeztél, anélkül, hogy elolvastad volna az előző fejezetet, akkor légyszives olvasd el az Lámpatípusok anatómiája fejezetet előbb.

Minden lámpának minimum kell egy Modul. Némelyiknek több is van. Például egy lámpának lehet egy pan attribútuma, de több tilt attribútuma (néhány kis fejű lámpa felfütykölve ugyanarra a vasra). Vagy elterjedtebb egy modul a pan, tilt, zoom, master fényerővel, és más közös funkciók és a másik modulon pedig minden egyes LED pixel egy wash mozgófejesben.

A Module Kezelőben, lehet szerkeszteni a modul információit. A kezelőt megnyithatod a Fixture Type Editor-ból. Így nézhet ki:

Vannak alul gombok amive modulokat lehet hozzáadni/törölni a meglévőkhöz. A Diagnostics gomb megnyitja a modulhoz tartozó figyelmeztetések/hibák ablakocskát.

A modul kezelő sorokban mutatja a modulokat és több oszlop van ami szerkeszti a modul elemet. Itt egy rövid leírás az oszlopokról:

  • No.:
    Ez egy automatikusan generált sorszám. Nem szerkeszthető.
  • Name:
    Ez a modul neve.
  • BeamAngle:
    Itt megadható egy statikus fénysugár szög. Ha lámpa variálható zoomal rendelkezik, akkor itt adj meg "1"-et és a fizikai zoom szögeket állítsd be a zoom attribútumnál.
  • BeamIntensity:
    Itt lehet megadni a fényerőt lumenben. Ezt használja az Ma 3D.
  • MIB Delay:
    a modulnak lehet saját MIB késési ideje. A "Default" értéket használja a MIB Delay időmeghatározás a Playback & MIB Timing beállítás a Setup-ban. Olvashatsz a MIB-ről a Munka a MIB-el fejezetben.
  • Class:
    A Class leírja a modul típusát. A cella szerkesztésekor megnyílik a Select Class ablakocska. Olvass róla lejebb.
  • Beam:
    A Beam opció meghatározza, hogy hogyan nézzen ki a fénysugár a 3D-ben. Szerkesztéskor megnyílik a Select Beam ablakocska:
    A lehetőségek: None, Spot, Wash, Effect, and Fiber. Vigyázz a LED osztály nem mutat fénysugarat - mindegy mit választotottál ki ebben a lehetőségben. Gyakran csak a Spot és Wash fénysugarak amiket rendesen mutat meg az MA 3D.
  • Model:
    A modell a 3D-s modell ami az MA 3D-ben lesz használva. A cellát szerkesztve, megnyílik a Model ablakocska. Ez megmutatja a show fájlban lévő modelleket. A saját tervezésű 3D modellekről olvasd el a 3D Modellezés és Beolvasás fejezetet az MA 3D leírásban.
  • Size X, Size Y, Size Z, Axis X, Axis Y, and Axis Z:
    A méret és tengely oszlopok leírják, az adott lámpa fizikai méreteit és a mozgástengelyei pozicíóját. Az oszlopot szerkesztve megnyílik a Body Description ablakocska. Ez másképp néz ki a kiválasztott osztálytól függően. A legtöbb osztálynak nincs test leírása és a cellákat nem lehet szerkeszteni.
  • Local:
    Ezt a cellát On-ra állítva, a modul helyben számolódik ki. Ezt nagy figyelemmel kell használni. Ezt általában a kövvető rendszereknél használják. segíthet ha van egy kis késés a követő rendszerben, a Színpadjelölő eszközöknél ha helyben számolja ki lehet némi előnye az egész rendszeren keresztüli számítással szemben. Ha viszont a helyi számolás sok eszköznél be van kapcsolva, akkor megfordulhat a dolog és még több késleltetés lehet a rendszerben.

 

Module Classes

The Class is used to describe the type of fixture. It defines how it is visualized in MA 3D. The Select Class pop-up opens when the cell is edited in the module manager.

This is a show description of the different classes:

  • None:
    This class does not display any 3D models in MA 3D.
  • Mirror:
    This class is for moving mirror fixtures. The default 3D model type is a classic "scanner" model.
  • Headmover:
    This is a moving head or moving body fixture type. The default 3D model type is a classic "yoke" model.
  • Conventional:
    This is used for non moving fixtures. The default 3D model type is a classic "parcan" model. This might also be a useful class for non-moving LED fixtures where the beam is to be visualized in MA 3D and the Stage window.
  • LED:
    This class is used for non moving LED fixtures. There is no light beam from this class. It is a light emitting surface. The default 3D model type is a "box" model.
  • Video Keystone:
    This class is used by some media servers. Nincs 3D-s model hozzárendelve az osztályhoz.
  • Video Effect Master:
    This class is used by some media servers. Nincs 3D-s model hozzárendelve az osztályhoz.
  • Video Layer:
    This class is used by some media servers. Nincs 3D-s model hozzárendelve az osztályhoz.
  • Video Output:
    This class is used by some media servers. Nincs 3D-s model hozzárendelve az osztályhoz.
  • Video Splitscreen:
    This class is used by some media servers. Nincs 3D-s model hozzárendelve az osztályhoz.
  • Moving Path:
    The moving path class is used by the Moving Path fixtures from MA Lighting. They can be used to move other 3D elements in MA 3D. Nincs 3D-s model hozzárendelve az osztályhoz.
  • Video Master:
    This class is used by some media servers. Nincs 3D-s model hozzárendelve az osztályhoz.
  • Grandma Bitmap Layer:
    This class is used by the Bitmap fixture from MA Lighting. This virtual fixture can be used to apply values to other fixtures - Read more in the Bitmap Fixture section. Nincs 3D-s model hozzárendelve az osztályhoz.
  • Marker:
    This class is used by the Stage Marker fixtures from MA Lighting. These virtual fixtures can be used to move a marker around the 3D space. Nincs 3D-s model hozzárendelve az osztályhoz.
  • Camera Controller:
    This class is used by the Camera Controller fixture from MA Lighting. This virtual fixture can be used to move a camera around the 3D space. Nincs 3D-s model hozzárendelve az osztályhoz.
  • Preset:
    This class is used by the auto generated "Fixture Type" fixtures that appear in the Fixture Sheet and Channel Sheet. It should not be used by other fixtures.
