Unlock Kulcsszó

To use to the Unlock keyword, type Unlock in the command line or use the shortcut UL / Unl.

Leírás

With the unlock keyword, you can unlock previous locked objects.

Információ:
A yellow lock indicates a locked object what can be unlocked.
A red lock indicates a locked object what is fixed by default. It can not be unlocked.


Locked Pool Objects

The Unlock keyword egy függvény kulcsszó.

Szintaxis

Unlock [Object-List]

Példák

1. Unlock cue 3 to edit the cue.

[Channel]> Unlock Cue 3

2. Unlock world 3 to edit the world.

[Channel]> Unlock World 3
