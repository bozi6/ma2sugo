Message Kulcsszó

A Message kulcsszó parancssorba való beírásához írja be a Message vagy röviden Mes a parancssorba.

Leírás

Az Message kulcsszó egy függvény kulcsszó, amely létrehozza az Üzeneteket a rendszerfigyelőben.

Ez hasznos, amikor exportálni akarod a rendszerfigyelőt mint .xml fájl és szeretnél bele írogatni jelölőket.

Szintaxis

Message "üzenet szövege"

Példa

  • A "test marker 1" üzenet megjelenítéséhez a rendszerfigyelőben, írd be a parancssorba:
[Channel]> Message "test marker 1"
