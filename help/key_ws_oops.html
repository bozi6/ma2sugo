Oops és undo

Az hoppá funkcióval, könnyen visszavonhatod a műveleteket.

Oops / vissza az utolsó akciót

Fontos:
Egy oopsed/undone akciót nem lehet visszafordítani.

Hogy oops / undo az utolsó akciót:

  1. Nyomd meg a Oops.
    A Visszavonás elfogadása ablakocska megnyílik.
  2. Bökj az Ok-ra vagy, nyomd meg a Please-t.

Az utolsó akció visszavonódott.

Oops/undo több akció visszavonása

Fontos:
Miután elmentetted a patch beállításait a Patch & Fixture táblázatban, az Oops lista újrakezdődik.
Fontos:
Egy oopsed/undone akciót nem lehet visszafordítani.

Tobb akció visszavonáas:

  1. tartsd nyomva az Oops gombot adig ameddig aMulti-Oops ablakocska megnyílik. Megjegyzés:
    Több akció kijelöléséhez, jelöld ki a legutolsót és onnan menj visszafelé.
    Nem lehet csak egy akciót kiválasztani a lista közepéről és csak azt visszavonni.
     
  2. Válaszd ki a visszavonandó akciót, rábökéssel vagy a képernyőkerék elforgatásával.
    A kiválasztott akciónak zöld a háttere. A kiválasztott visszavonandó akciók száma látszik a bal alsó sarokban lévő gombon.
  3. Bökj az Undo x Actionsgombra.

A kiválasztott akciók visszavonódnak.

oops beállításait

Alapvetően, minden következő akció bekerül az oops listájába:

  • General (Beállítások)
  • Views (Nézet változtatásaok)
  • Prog (Programozó akciók)

Testreszabható az oops beállításoknál. Például csak a programozó beállításait akarod hozzáadni, és nem minden nézetváltás, deaktiváld a view beállításait az oops beállításnál.

A hoppácska beállításai:

  1. Nyomd meg a Setup, bökj a User, és böjk a Oops Settings.-re
    A Oops Settings menü megjelenik.
  1. Bökj a beállításra amit be/ki akarsz kapcsolni.

A hoppácska beállítások megváltoznak és elmentődnek a felhasználói profilba.

Visszatérés egy előző mentett show változathoz

Every time you saved the show, the previous state of the show file will be saved as a .backup file. The grandMA2 creates up to ten .backup files. After the tenth .backup file is created, the first .backup file will be deleted. This allows to revert a previously saved show file version.

  1. Press Backup.
    The Backup menu opens.
  2. Select the drive tab and tap at Load Show.

 

  2. Tap at the view button until Viewing Shows & Backups or Viewing Backups appears.
    Now, you can see all saved shows as .show.gz file and their backups as .backup file.
  3. Tap at the .backup file you want to revert and select the data you want to load.

    Tipp:
    If you are not sure what data you have changed, we recommend to load all data. Tap Check All.
     
  4. Tap Load Show.

 

 The selected backup is reverted.
