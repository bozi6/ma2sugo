EffectPhase Kulcsszó

To get the EffectPhase kulcsszó beírásához, írd be EffectPhase or EffectP a parancssorba.

Leírás

EffectPhase is a helping keyword to indicate individual effect phases.

Szintaxis

EffectPhase

EffectPhase [Value-list]

Példák

  • To switch the encoders to adjust the effect phase layer írd be a parancssorba:
[Channel]> EffectPhase

 

  • To knock out any effect phase values of the programmer for the selection/feature írd be a parancssorba:
[Channel]> Off EffectPhase

 

  • To set the effect phase for the selection/feature to 180 degrees írd be a parancssorba:
[Channel]> EffectPhase 180

 

  • To Set the effect phase for "pan" for the selection to 180 degrees írd be a parancssorba:
[Channel]> Attribute "pan" At EffectPhase 180
