Protocol Kulcsszó

A Protocol kulcsszóhoz, írd be Protocol a parancssorba, vagy rövidebben Prot.

Leírás

Ez egy objektum kulcsszó, hogy elérd a hálózati protokollokat.

Mentheted, vagy törölheted a hálózati protokollokat, vagy hozzárendelhetsz paramétereket.

További információk: Hálózati DMX Protokollok.

ETC Net2, Pathport TTL 0...255 TTL = Time to live

Rendeljen hozzá olyan útválasztó (ugrások) számot, amelyeken a multicast forgalom megengedett áthaladni, mielõtt lejár a hálózaton.
