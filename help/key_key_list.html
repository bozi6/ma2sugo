List Gomb - Listázás Gomb

A List gomb a parancsterületen van balra a Store gombtól.

 

  • Megnyomva a List gombot, az aktuális gomboldalon megjelenik az executor gombok 201-220 X-Key címkéi.
     

 

  • Nyomva tartva a  + List gombokat, beíródik a List kulcsszó a parancssorba.

Ideiglenes tároló megnyitása

A következõ gyorsbillentyûkkel megnyílik az ideiglenes tároló a második képrenyõn (kivétel agrandMA2 onPC - MA onPC wing nélkül).

Az ideiglenes tárolók könnyen azonosíthatóak, mert piros keret veszi körül õket.

  • Nyomva tartva a List + Channel gombot, megnyílik az ideiglenes csatorna tároló.
     

 

[time] : Macro: Menu ChannelPage

 

  • Nyomva tartva a List + Group gombot, megnyílik az ideiglenes csoport tároló.
     

 

[time] : Macro: Menu Group

 

  • Nyomva tartva a List + Sequence gombot, megnyílik az ideiglenes szekvencia tároló.
     

 

[time] : Macro: Menu Sequence

 

  • Nyomva tartva a List + Page gombot, megnyílik az ideiglenes executor tároló.
     
     
[time] : Macro: Menu Page

 

  • Nyomva tartva a List + Macro gombot, megnyílik az ideiglenes makró tároló.
     

 

[time] : Macro: Menu Macro

 

  • Nyomva tartva a List + Preset gombot, megnyílik az ideiglenes preset tároló, a preset vezérlõ által kiválasztott típusból.
     

 

[time] : Macro: Menu Preset

 

  • Nyomva tartva a List + View gombot, megnyílik az ideiglenes nézet tároló.
     

 

[time] : Macro: Menu View

 

  • Nyomva tartva a List + Effect gombot, megnyílik az ideiglenes effekt tároló.
     

 

[time] : Macro: Menu Effect
