Cut Kulcsszó

To enter the Cut keyword into the command line, type Cut into the command line.

Leírás

Cut is a function used to specify the source objects for a two-step move action.

Szintaxis

Cut [Object-list]

The given object list is temporarily stored for later use as source objects for the following Paste command.

For more information on cut and paste see Paste. 

Fontos:
Cut & Paste does not work in cue objects. To move cues, use the Move keyword.

Példa

 

[Channel]> Cut Preset 4.1

 

Prepares color preset 1 to be moved via the Paste command.
