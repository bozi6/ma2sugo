Goto Kulcsszó

To get the Goto keyword in the command line press Goto.

Another way is to type Goto vagy rövidenGot a parancssorba.

Leírás

Goto egy függvény kulcsszó to jump in a list. Set the fade time for Goto in the Setup I Show I Playback & MIB Timing I Goto.

Szintaxis

Goto [Cue object] [Executor object] / [Opció] = [Opció Érték]

Opciók

To get a list of all available options to the Goto keyword írd be a parancssorba:

[Channel]> Goto /?

The Goto keyword has the following options:

Opció Option Shortcut Opció értéke Option Value Shortcut Leírás
cue_mode cm Normal N Normal Goto mode.
cue_mode cm Assert A Goto with assert in the original timing.
cue_mode cm XAssert XA Goto with assert in the cuetiming of current cue.
cue_mode cm Release R Goto finishes the fade-in and switches the executor off.
userprofile upr "Name of the user profile", e.g. "Klaus" not available Normal Goto mode with specified user profile.

Példák

  • To go to cue 3 of the selected executor írd be a parancssorba:
[Channel]> Goto Cue 3

 

  • To go to cue 5 of executor 4 írd be a parancssorba:
[Channel]> Goto Cue 5 Executor 4
