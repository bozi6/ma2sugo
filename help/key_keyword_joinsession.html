JoinSession Kulcsszó

To get the JoinSession kulcsszó beírásához, írd be JoinSession or one of the shortcuts J or JS a parancssorba.

Leírás

JoinSession is a function used to join or create a session. If no option is given the recent entered settings in the MA Network Control will be used.

Szintaxis

JoinSession [Opció Érték]

Opciók

To get a list of all available options to the JoinSession keyword, írd be a parancssorba:

[Channel]> JoinSession /?

The JoinSession keyword has the following options:

Opció Option Shortcut

Option Value Examples

 Leírás
password pw "1234" Enters the session password.
sessionnumber s "1" Enters the session number.
sessionname n "FOH" Enters the session name.

Példa

To join the session called FOH with the session name and the password 1235 írd be a parancssorba:

[Channel]> JoinSession 1 "FOH" /pw = "1235"
