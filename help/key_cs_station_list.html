[IP-cím]

[IP-cím] Thru [IP-cím]

[Host-ID]

[Host-ID] Thru [Host-ID]

 

Fontos:
Az IP cím egy 32 bites pontozott decimális jelölés, amely négy számot tartalmaz, 0 és 255 között, és pontokkal elválasztva, például 192.168.0.101.

 

  • A host ID az IP-cím egyedi része a hálózaton belül. A gazdagép-azonosító általában az IP-cím utolsó három tizedesjegyét tartalmazza. A 192.168.0.101 IP-cím gazdagép-azonosítója 101, a hálózati azonosító pedig 192.168.0. 
  • Ha a Thru parancs eleje vagy vége hiányzik, akkor az elsõ vagy az utolsó elõfordulást kell használni. 

 

Példa:

[Channel]> SetIP ethO 192.168.0.101

Beállítja az IP címet. 

 [Channel]> Shutdown 1 Thru 4

 Minden eszközt leállít aminek a host ID-je 1, 2, 3, és 4. 

 

További információk és hogyan állítsd be az IP címet: SetIP.
