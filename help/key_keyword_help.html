Help Kulcsszó

A Help kulcsszó beírásához a parancssorba, nyomd meg a Help gombot.

Máésik módja az, ha beírod a Help vagy röviden H közvetlenül a parancssorba.

Leírás

A Help egy függvény kulcsszó, ami megnyitja a súgó ablakot.

Szintaxis

Help

Help [kulcsszó]

Help "Keresett szöveg"

Példák

  • a súgó ablak megnyitásához írd be a parancssorba :
[Channel]> Help

 

  • A Fixture kulcsszó súgójának megjelenítéséhez írd be a parancssorba:
[Channel]> Help Fixture
