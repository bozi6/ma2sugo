Mentés mint grandMA3

Fontos:
Az opció "Export Patch to grandMA3" csak az grandMA2 onPCvel mûködik.
Ha a Export Patch to grandMA3 gomb nem látszik, akkor a rendszer nem felel meg teljesen ennek a követelménynek. További információk rendszerigények.

Egy grandMA2 show elmentése mint grandMA3 show file meg kell felelnie a következõ dolgoknak:

  • Fixture Types
  • Attributes
  • Fixtures

Szükséges: Egy létrehozott grandMA2 show fájl.

A megnyitott grandMA2 show fájl mentéséhez, mint grandMA3 show fájl, kövesd a következõ lépéseket.

  1. A Backup menü megnyitásához, bökj a Backup gombra, vagy nyomd meg a Backup gombot.
    A Backup menü megnyílik.
  2. Bökj az Export Patch to grandMA3-ra.

Az aktuális grandMA2 show file elmentõdik a C:\ProgramData\MALightingTechnology\gma3_x.x\shared\shows mappába.

Most, ezt a show file-t meg lehet nyitni a grandMA3 szoftverben.
