Save as grandMA3

Fontos:
The option "Export Patch to grandMA3" works only on a grandMA2 onPC.
If the button Export Patch to grandMA3 is not visible, the system requirements for this feature are not fulfilled. For more information, see system requirements.

A grandMA2 show file saved as a grandMA3 show file exports the following elements:

  • Fixture Types
  • Attributes
  • Fixtures

Requirements: A created grandMA2 show file.

To save the currently open grandMA2 show file as a grandMA3 show file, follow the steps below.

  1. To open the Backup menu, tap Backup or press Backup.
    The Backup menu opens.
  2. Tap Export Patch to grandMA3.

The current grandMA2 show file is saved in C:\ProgramData\MALightingTechnology\gma3_x.x\shared\shows.

Now, this show file can be opened in the grandMA3 software.
