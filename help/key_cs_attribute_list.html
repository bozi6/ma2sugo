Attibútum Lista

Az Attibútum Lista egy objektum az atribútumok listájával. 

Syntax

Attribute [list]

Feature [list]

PresetType [list]

 

Fontos:
Az attribútumlista szintaxisa megegyezik az objektumlistával. Minden objektumtípus az objektumok attribútumára van bontva. Ezek az attribútumobjektumok alkotják az attribútumlistát.

 

Példa:

[Channel]> On Feature "GOBO1"

Aktiválja az attribútumot a "GOBO1" képességben. 

[Channel]> Off Attribute 8.1.1 Thru 4

Kiüti az elsõ 4 késattribútumot a programozóból.
