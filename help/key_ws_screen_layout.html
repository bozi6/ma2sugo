Képernyő elrendezés

Ez a fejezet bemutatja a képernyő elrendezéseket a különböző képernyőkön.

1. képernyő

AZ 1. képernyő vagy parancsképernyő három területből áll, alulról felfelé a következőképpen néz ki

  1. X-Key Címkék to label X-Keys.
  2. Parancssor parancsok beírása, visszajelzés megnyitása vagy üzenetközpont megnyitása.
  3. Felhasználó Által Meghatározott Terület ablakok létrehozása és elrendezése.

2. képernyő

A 2. képernyő hét részből áll.

  1. Master Részleg mutatja a kiválasztott executort.
  2. User Defined Area ablakok létrehozása és elrendezése.
  3. Parancssor parancsok beírása, visszajelzés megnyitása vagy üzenetközpont megnyitása.
  4. Preset Vezérlő sáv preset választásához.
  5. Kódoló eszközsáv tulajdonságok vezérléséhez.
  6. Idő vezérlése egy program idejének és végrehajtási idejének álítására.
  7. Nézet Gombok előzőleg beállított elrendezés visszaállítása.

3. és 4. képernyők (csak a grandMA2 light vagy grandMA2 full-size nál)

3. és 4. képernyők három részből állnak.

  1. Mini Executor Sáv információk az executor tartalmáról és funkciójáról.
  2. User defined area ablakok létrehozása és elrendezése.
  3. Nézet Gombok előzőleg beállított elrendezés visszaállítása.

5. és 6. képernyők (külső képernyők)

A külső képernyők két részből állnak.

  1. User defined area ablakok létrehozása és elrendezése.
  2. Nézet Gombok előzőleg beállított elrendezés visszaállítása.
