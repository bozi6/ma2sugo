További rendszereszközök

Ez egy gyûjteménye a rendszerablakoknak, amiket máshol nem írtunk le. Ezek nem lapok, tárolók, vagy presetek, de más ablakok amik néhány rendszerszintû beállításokat kezelnek.

A fejezetek a következők:

  • Message center - Üzenet központ
  • Help - Súgó
  • Clocks - Órák
  • Sound input window - Hangbemenet ablak
  • VPU pixel mapper view - VPU pixel leképező
  • Network dimmer - Hálózati dimmer
  • Desk status - Pultállapot
  • Performance window - Teljesítményablak
  • System monitor - Rendszermonitor

Az ablakok eléréséhez, bökj egy üres részre a képernyőn és bökj az Other vagy System fülre.
