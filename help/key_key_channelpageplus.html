Ch Pg + (Channel Page) Gomb - Csatorna Oldal Gomb

A Ch Pg + gomb balra található a 100mm húzónál.

 

  • Megnyomva a Ch Pg + gombot, végrehajtja a ChannelPage parancsot. A csatorna oldalak a 3. és 4. képernyőn látszanak.

 

 

  • Megnyomva a Ch Pg + ismét, a következő csatorna oldalra ugrik.
     
  • Ha a Link Fadersopció a lámpa lapon aktív, akkor a  Ch Pg + átvált a dinamikus csatorna oldalra a 3. és 4. képernyőn.
