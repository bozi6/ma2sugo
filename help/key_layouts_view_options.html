Alaprajz Nézet Opciók

Opciók megnyitása

  1. Az alaprajz nézet beállításainak megnyitásához, bökj a  -ra a bal felső sarokban. 
  2. Az opciók ablak megnyílik. 
  1. A különböző opciók közötti váltáshoz, az alaprajz nézet fejléce alatt, bökj a megfelelő fülre . 

Tároló Elemek Alapértelmezése

Fontos:
A Pool Element Default és Fixt./Chan. Defaults fülek alatti beállítások a következő elem hozzáadásakor lesznek érvényesek.
  1. A tároló elemeinek méretezéséhez, bökj a Size X és Size Y gombok alá. 
    -A számológép megnyílik.
    -Írd be az elemek méretét.
  2. Utána, állítsd be a betűméretet:
    -Bökj a Font Size-Betűméret legördülő listára.
    -A  Select Font Size-válassz betűméretet megnyílik.
    -Most, bökj a háromból egy betűméretre. 
  3. Az ID, név, vagy típus mutatásához/elrejtéséhez az alaprajz nézetben, bökj a Show ID, Show Name vagy Show Typegombok alá. 
  4. Miután beálítottad, állítsd be a háttérszínt, keretszínt egy elemhez:
    -Bökj a Background Color, Border Color és Text Colorgombok alá. 
    -A Select Color-válassz színt megnyílik.
    -Most állítsd be a megjelenő színeket és válassz a Predefined Colors-Előre beállított színek, Swatch Book-Színminta könyv, HSB és Fader-húzó. 
    -A szín megváltoztatásához, bökj a színkörbe vagy bökj a százalékos gombokra a diagram alatt..
    -A kiválasztott szín elfogadásához, bökj a Please-re.
    -A színek visszaállításához, bökj a Reset vagy Reset to White gombra. 
  5. Utána, állítsd be a szimbólumok megjelenését:
    -Bökj a legördülő gombra a Symbol alatt, a Select Symbol-válassz szimbólumot megnyílik.
    -Most, bökj a három lehetőségből egyre.
  6. A dimmer sáv és dimmer érték megjelenítéséhez, bökj a Show Dimmer Bar és Show Dimmer Value gombokra.
  7. Utána, válaszd ki a kép méretét:
    -Bökj a legördülő lista gombra az Image Size alatt.
    -A Select Image Size -válassz képméretet megnyílik.
    -Válassz egyet a három képméret közül.
  8. Az objektumok megjelenítésének megváltoztatása:
    -Bökj a legördülő lista gombra a Visualization alatt.
    -A Select Visualization-válassz megjelenést megnyílik.
    -Bökj valamelyik opcióra.

Most beállítottad az alapvető tárolóelemeket az aktuális alaprajzon, kedvedre.

Fontos:
Ha az új alaprajzokon is ezeket a baállításokat szeretnéd használni, akkor bökj a Save to Default a Layout Optionsfejlécében. 

Fixt. / Chan. Defaults - Lámpa/csatorna alapértelemezések

Az alapvető beállítások megegyeznek a lámpáknál/csatornáknál ezeknél a tároló elemeknél. Lásd Tároló Elemek Alapértékei, hogy képbe kerülj. Azonban, van néhány különbség itt. 

  1. A lámpa/csatorna ID jének elrejtéséhez/megjelenítéséhez bökj a Show Fixture ID vagy Show Channel IDgomb alá.
  2. Következő, az RDM értesítések ki/bekapcsolásához az alaprajz nézetben:
    -Bökj az  RDM Notificationsgomb alá.
    Ha engedélyeztet az RDM értesítéseket, akkor az RDM risztás vagy figyelmeztetés látszani fog az alaprajz nézetben. Bővebben az RDM értesítésekről: RDM értesítések beállítása.

Layout Data - Alaprajz Adatok

Az alaprajz adatok egy alaprajzra vonatkoznak csak.

  1. A rácsok méretének állítása:
    -A rács szélessége, magassága, vagy igazításához, bökj To adjust the width and length of the grid or the snap grid, bökja az Grid X és Grid Y vagy Snap Grid X és Snap Grid Y  gombok alá.
    -A számológép megnyílik.
    -Most, type the size of of the corresponding grid.
Tipp:
The grid stays visible whenever you edit its size. To make the grid invisible again, set one of the axes to zero. 
  1. Enable or disable the snap grid:
    -It is possible for the snap grid to stay always visible whenever moving an object.
    -To do so, tap the button below Snap Always to toggle it to Yes.
  2.  After setting the grid, select the background color:
    -Tap the button below Background.
    -The pop-up Select Color opens.
    -To manage the color diagram, see Pool Element Defaults.
  3. To display the markers in the fixtures and channels, toggle the button below Marker to Yes.
  4. To turn on the fast call, toggle the button below Fast Call to Yes.
Tipp:
If the fast call is enabled in the tab Title Buttons, the button Fast Call will be highlighted in red in the title bar of the layout view. 
  1. Next, select the playback you would like to use:
    -Tap the button below Pool Playback.
    -The pop-up Select Pool Playback opens. For more information on playback see Cues and sequences – Playing back cues.
     
    -Select one option here. 
  2. Then, select a layer by tapping the drop-down button below Select Layer.
    The pop-up Select Layer opens. For more information on layers see Layers in sheets – channel sheet and fixture sheet.

     
    -Select one of the three layers here. 
  3. To visualize the effects Gobo and Color even if the dimmer is closed, tap the button below Symbols Features and toggle it to Always.
    If the features are to comply with the dimmer, tap and toggle to with Dimmer. 

Tools

The following settings are applied to the current layout view. 

  1. To reverse the respective selection – select or move – enable the button below Double tap mode:
    Tap the exact same spot twice and then move your finger either to move the screen or to select the objects. 
  2. Next, set the cue colors:
    -Bökj a legördülő lista gombra a Cue Colors.
    -The pop-up Select Cue Colors opens.
  • Off: Does not display cue colors.​​
  • On: Displays the cue colors of a selected executor. 
  • My Exec: Displays the cue colors of the executor you have activated. 
  1. To display the zoom bar on the right of the layout view, enable the button below Zoom Bar visible.
  2. To use the screen encoder for zooming in and out of the layout view, enable the button below Zoom Encoder.
  3. Enable the horizontal and vertical scroll bar by tapping on the corresponding buttons below ScrollBar horizontal and ScrollBar vertical. 
  4. To enable ruler, tap the button below Rulers visible.
Tipp:
When adjusting the grid the ruler proves itself to be very useful. 
  1. Choose an edit mode to select certain objects:
    -Bökj a legördülő lista gombra a Select Edit Mode.
    -The pop-up Select Edit Mode opens.

Title Buttons

Tap the button you would like to be displayed in the title bar of the layout view.

  • Enabled buttons have a yellow check mark and are highlighted in yellow. 
  • Disabled buttons have a black background. 

Close the Options

After adjusting the options to your liking tap  in the upper right corner of the dialog. The window Layout Options closes. 

To close the layout view altogether, tap Delete Window.
