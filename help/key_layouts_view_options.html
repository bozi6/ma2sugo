Layout View Options - Alaprajz Nézet Opciók

Az opciók megnyitása

  1. Az alaprajz nézet opcióinak megnyitásához, bökj a  bal felső sarkába a nézetnek. 

    2. Alaprajz Nézet Opciók

    Opciók megnyitása

    1. Az alaprajz nézet beállításainak megnyitásához, bökj a  -ra a bal felső sarokban. 
    2. Az opciók ablak megnyílik. 
    1. A különböző funkciók közötti navigáláshoz, bökj a megfelelő fülre a fejléc alatti részre. 

    Tároló elemek alapbeállítása

    Fontos:
    A beállítások amiket a Pool Element Default és a Fixt./Chan. Defaultsban végrehajtasz, az ezt követően hozzáadott elemekre lesznek érvényesek.
    1. A tároló elem méretének megváltoztatásához, bökj a Size X és Size Y gombokra.. 
      -A számológép megnyílik.
      -Írd be az elemek méretét.
    2. Utána állítsd be a betűtípus méretét:
      -Bökj a Font Sizelegördülő elemre.
      -A Select Font Size ablakocska megnyílik.
      -Most, válaszd ki a három betűméret közül az egyiket. 
    3. Az elem azonosítójának, nevének vagy típusának engedélyezéséhez vagy letiltásához az elrendezés nézetben bökj az alábbi gombokra: Show ID, Show Name vagy show Type. 
    4. Ezek után, állítsd be a háttérszínt, keretszínt és szövegszínt egy elemre:
      -Bökj a következő gombokra lejjebb Background Color, Border Color és Text Color. 
      -A Select Color-válassz színt- ablakocska megnyílik.
      -Most, beállíthatod a színeket és rábökéssel választhatsz a Predefined Colors-előre definbiált szín, Swatch Book-színminta könyv, HSB és Fader-húzók között. 
      -A szín változtatáshoz, bökj a színkörre, vagy bökj egy diagram alatti gombra a százalékos értékkel..
      -a szín alkalmazásához, bökj a Please-re.
      -A szín visszaállításához, bökj a Reset or Reset to White-visszaállítás fehérre. 
    5. Utána állítsd be a megjeleníteni kívánt szimbólumot:
      -Bökj a legördülő menüre a Symbol alatt, és a Select Symbol-válassz szimbólumot megnyílik.
      -Most, bökj a három lehetőség közül egyre.
    6. A dimmersáv megjelenítésénwk ki/bekapcsolásához, bökj a Show Dimmer Bar-dimmersáv mutatása és Show Dimmer Value-dimmer érték mutatása gombokra.
    7. Utána válaszd ki a kép méretét:
      -bökj, az Image Sizelegördülő lista alá.
      -ASelect Image Size ablakocska megnyílik.
      -Válassz egyet a három lehetőségből.
    8. Válasszd az objektumok megjelenítését:
      -Bökj a Visualization-megjelenítés gombra.
      -A Select Visualization ablakocska megnyílik.
      -Válassz ki egyet a lehetőségek közül.

    Most beállítottad az alapvető elemek megjelénését az aktuális alaprajzon.

    Fontos:
    Ha ezeket a beállításokat szeretnéd az új alaprajzokra is alkalmazni, bökj a Save to Default-ra a Layout Options fejlécében. 
    1. A különböző opciók közötti váltáshoz, az alaprajz nézet fejléce alatt, bökj a megfelelő fülre . 

    Tároló Elemek Alapértelmezése

    Fontos:
    A Pool Element Default és Fixt./Chan. Defaults fülek alatti beállítások a következő elem hozzáadásakor lesznek érvényesek.
    1. A tároló elemeinek méretezéséhez, bökj a Size X és Size Y gombok alá. 
      -A számológép megnyílik.
      -Írd be az elemek méretét.
    2. Utána, állítsd be a betűméretet:
      -Bökj a Font Size-Betűméret legördülő listára.
      -A  Select Font Size-válassz betűméretet megnyílik.
      -Most, bökj a háromból egy betűméretre. 
    3. Az ID, név, vagy típus mutatásához/elrejtéséhez az alaprajz nézetben, bökj a Show ID, Show Name vagy Show Typegombok alá. 
    4. Miután beálítottad, állítsd be a háttérszínt, keretszínt egy elemhez:
      -Bökj a Background Color, Border Color és Text Colorgombok alá. 
      -A Select Color-válassz színt megnyílik.
      -Most állítsd be a megjelenő színeket és válassz a Predefined Colors-Előre beállított színek, Swatch Book-Színminta könyv, HSB és Fader-húzó között. 
      -A szín megváltoztatásához, bökj a színkörbe vagy bökj a százalékos gombokra a diagram alatt..
      -A kiválasztott szín elfogadásához, bökj a Please-re.
      -A színek visszaállításához, bökj a Reset vagy Reset to White gombra. 
    5. Utána, állítsd be a szimbólumok megjelenését:
      -Bökj a legördülő gombra a Symbol alatt, a Select Symbol-válassz szimbólumot megnyílik.
      -Most, bökj a három lehetőségből egyre.
    6. A dimmer sáv és dimmer érték megjelenítéséhez, bökj a Show Dimmer Bar és Show Dimmer Value gombokra.
    7. Utána, válaszd ki a kép méretét:
      -Bökj a legördülő lista gombra az Image Size alatt.
      -A Select Image Size -válassz képméretet megnyílik.
      -Válassz egyet a három képméret közül.
    8. Az objektumok megjelenítésének megváltoztatása:
      -Bökj a legördülő lista gombra a Visualization alatt.
      -A Select Visualization-válassz megjelenést megnyílik.
      -Bökj valamelyik opcióra.

    Most beállítottad az alapvető tárolóelemeket az aktuális alaprajzon, kedvedre.

    Fontos:
    Ha az új alaprajzokon is ezeket a baállításokat szeretnéd használni, akkor bökj a Save to Default a Layout Optionsfejlécében.

    Fixt. / Chan. Defaults - Lámpa/csatorna alapértelemezések

    Az alapvető opciók megegyeznek a csatornák/lámpáknál ebben a tárolóelemben. Lásd Tároló eklemek alapjai. Azonban, van néhány különbség is. 

    1. A lámpa/csatorna ID megjelenítésének ki/bekapcsolásához, bökj a Show Fixture ID vagy Show Channel ID gomb alá.
    2. Utána, engedélyezd/tiltsd az RDM üzenetek megjelenítését az alaprajz nézetben:
      -Bökj az RDM Notifications gomb alá.
      Ha engedélyezted az RDM üzeneteket, akkor az RDM riasztások vagy figyelmeztetések meg fognak jelenni az alaprajz nézetben is. További információkért, lásd RDM üzenetek beállítása.

    Alaprajz adatai

    Az alaprajzi adatok cask egy elrendezésre vonatkoznak.

    1. A rácsméret beállítása:
      -A rács és a rácsraigazítás szélességének és magasságának állításához, bökj valamelyik gomb alá a Grid X és Grid Y vagy a Snap Grid X és Snap Grid Y közül. 
      -A számológép megnyílik.
      -Most, beírhatod a megfelelő méreteket.

      2. Az alapvető beállítások megegyeznek a lámpáknál/csatornáknál ezeknél a tároló elemeknél. Lásd Tároló Elemek Alapértékei, hogy képbe kerülj. Azonban, van néhány különbség itt. 

      1. A lámpa/csatorna ID jének elrejtéséhez/megjelenítéséhez bökj a Show Fixture ID vagy Show Channel IDgomb alá.
      2. Következő, az RDM értesítések ki/bekapcsolásához az alaprajz nézetben:
        -Bökj az  RDM Notificationsgomb alá.
        Ha engedélyeztet az RDM értesítéseket, akkor az RDM risztás vagy figyelmeztetés látszani fog az alaprajz nézetben. Bővebben az RDM értesítésekről: RDM értesítések beállítása.

      Layout Data - Alaprajz Adatok

      Az alaprajz adatok egy alaprajzra vonatkoznak csak.

      1. A rácsok méretének állítása:
        -A rács szélessége, magassága, vagy igazításához, bökj To adjust the width and length of the grid or the snap grid, bökja az Grid X és Grid Y vagy Snap Grid X és Snap Grid Y  gombok alá.
        -A számológép megnyílik.
        -Most, írd be a méretét a megfelelő rácsnak.
      Tipp:
      A rács mindaddig látszik ameddig a méretét állítod. A rács láthatatlanná tevéséhez, állítsd be az egyik tengelyt nullára. 
      1. A rácsraigazítás ki/bekapcsolása:
        -Lehetséges objektumok mozgatásához mindig láthatóvá tenni a rácsraigazítást.
        -Ehhez, bökj a Snap Always gomb alá, az átkapcsoláshoz Yes legyen kiválasztva.
      2.  A rács beállítása után, jöhet a háttérszín:
        -Bökj a Background gomb alá.
        -A Select Color-színválasztó ablakocska megnyílik.
        -A színdiagram állításához, lásd Tároló elemek alapjai.
      3. A jelölők mutatása a csatornák/lámpák esetében, állítsd a Marker gomb alatt Yes-re.
      4. A fast call-gyorshívás beállításához, állítod a Fast Call gomb alatt Yes-re.
      Tipp:
      Ha a gyorshívás engedélyezett a Title Buttons fülön, A Fast Call gomb pirossal van kiemelve az alaprajz nézeten. 
      1. Ezután válaszd ki a lejátszást amit használni szeretnél:
        -Bökj a Pool Playback-Tároló lejátszásagomb alá.
        -A Select Pool Playback-válassz tároló lejátszást ablakocska megnyílik. További információ a lejátszásról Képek és szekvenciák - képek visszajátszása fejezetben.
         
        -Válassz egyyet a lehetőségekből. 
      2. Ezután válassz egy rétegek, a Select Layer legördülő gombból.
        A Rétegválasztás ablakocska megnyílik. További információ a rétegekről Rétegek a lapokon - csatorna/lámpa lapok résznél.

         
        -Válassz egyet a három réteg közül. 
      3. A Gobok és színek megjelenítéséhez, akkor is ha a dimmer zárva van, bökj a Symbols Features gomb alá és állítsd át Always-ra.
        Ha ezt a dimmerrel együtt szeretnéd, akkor bökj a with Dimmer-re. 

      Tools-Eszközök

      A következő beállítások, az aktuális álaprajz nézetre vonatkoznak. 

      1. A választás megfordításához -select vagy move - engedélyezd a Double tap modegomb alatt:
        Bökj kétszer pontosan ugyanoda, és utána ha mozgatod az ujjad akkor váltasz gyorsan a kiválasztás/mozgatás között. 
      2. Ezután, állítsuk be a kép színeit:
        -Bökj a legördülő listára a Cue Colors-kép színe gomb alatt.
        -A Select Cue Colors-Válassz kép színt ablakocska megnyílik.
      3. A kiválasztás megfordításához - select vagy move - válaszd ki a Double tap modegomb alatt:
        Ha ugyanoda böksz kétszer akkor az ujjad mozgatásával, mozgathatod az alaprajzot. 
      4. Ezután, állítsuk be a kép színeit:
        -Bökj a legördülő lista gombra a Cue Colors.
        -A Select Cue Colors-válassz kép színt ablakocska megnyílik.
      • Off: Nem mutatja a kép színeit.​​
      • On: A kiválasztott executor kép színeit mutatja. 
      • My Exec: Az aktív executor kép színeit mutatja. 
      1. a nagyítósáv mutatásához, engedélyezd a Zoom Bar visiblegomb alatt.
      2. Ha a képernyőkódolót az elrendezés nézetének nagyításához és kicsinyítéséhez szeretnéd használni, engedélyezd a Zoom encoder alatti gombot.
      3. A függőleges és vízszintes görgetősávok engedélyezéséhez, a megfelelő felirat alatti gombra bökj rá ScrollBar horizontal-görgetősáv vízszintes és ScrollBar vertical-görgetősáv függőleges. 
      4. A vonalzó engedélyezéséhez, bökj a Rulers visiblegomb alá.
      Tipp:
      A rács állításakor a vonalzó nagy segítség lehet. 
      1. Válassz szerkesztési módot bizonyos objektumok kiválasztásához:
        -bökj a legördülő listára a Select Edit Modegomb alatt.
        -A Select Edit Mode ablakocska megnyílik.

      Fejléc gombok

      Bökj arra a gombra, amelyiket szeretnéd, hogy megjelenjen az alaprajz nézet fejlécén.

      • Az engedélyezett gombokon sárga pipa és sárga kiemelés található. 
      • A letiltott gomboknak fekete a háttere. 

      Az opciók bezárása

      ha a beállítások elneyerték tetszésedet, bökj a  gombra az ablak jobb felső sarkában. A Layout Options ablak bezáródik. 

      Az alaprajz nézet bezárásához, bökj a Delete Window-ra.
