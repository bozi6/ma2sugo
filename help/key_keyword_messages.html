Messages Kulcsszó

A Message kulcsszóhoz, írd be a Messages -t a parancssorba.

Leírás

A Messages kulcsszóval lehetséges

  • üzeneteket listázni a parancssori visszajelző ablakba.
  • üzeneteket exportálni a messages.xml fájlba.
  • üzeneteket törölni az üzenetközpontból, például az előző show fájlokból.

Több információ a fájlok exportálásához általános export funkciók.

A Messages kulcsszó egy objektum kulcsszó.

Szintaxis

[Funkció] Üzenet / Feltétel = [Feltétel] [Operation Syntax] [Feltétel Érték]

Példák

  • Minden üzenet listázása a parancssori visszajelző ablakba, a kondíció típusával és az érték = riasztással.
[Channel]> List Messages /cnd = "type = 'alert'"

 

  •  Minden figyelmeztetés, risztás, és hibaüzenet exportálása, 12:45 előtt a messages.xml fájlba.
[Channel]> Export Messages /cnd="(type >= 'warning' or time <= '12:45:00') and !(new = 'true')"

 

  • Minden risztás üzenet törlése az üzenetközpontból.
[Channel]> Delete Messages /condition="type = 'alert'"
