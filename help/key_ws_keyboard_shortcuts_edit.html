Gyorsbillentyûk szerkesztése

Fontos:
A gyorsbillentyûk csak a grandMA2 onPC-ben mûködnek.

Szerkesztheted az alapbeállítást, és testre szabhatod, ahogyan tetszik.

  1. Kattints a Setup és a console fül alatt, kattints az Edit Keyboard Shortcuts gombra.
    A Gyorsbillentyûk szerkesztése ablak megnyílik.
  2. Keresd meg az a kombinációt, amit szerkeszteni szeretnél, jobb gombbal kattints rá a cellára amelyik a Hardkey oszlopban van, vagy a cellára a PCkey oszlopban.
    Ha rákattintottál a Hardkey oszlopra, a Change Direct Access Shortcut ablakocska megnyílik. Információ:
    Azok a billentyûk, amelyek már vannak hozzárendelve egy másik virtuális billentyûzetkódhoz, már nem szerepelnek a listában, és fordítva. Ha olyan billentyûre vagy virtuális billentyûzetkódra van szüksége, amely nincs felsorolva, módosítania vagy törölnie kell a billentyûparancsot, ahol használatos. Ha a billentyû vagy a virtuális billentyûzet szabad, hozzárendelheti.

    Ha a PCkey oszlopra kattintottál, akkor a Select Virtual Key Code ablakocska nyílik meg.
  3. Kattints a Hardkey vagy PCkey gombra amelyiket szeretnéd.
    A Change Direct Access Shortcut vagy Select Virtual Key Code ablakocska bezáródik.

A billentyûparancsok amiket szerkesztettél, elmentõdnek a show fájlba.
