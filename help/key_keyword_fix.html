Fix Kulcsszó

To get the Fix keyword in the command line press Fix.

You can also type Fix directly a parancssorba.

Leírás

Fix egy függvény kulcsszó to fix objects on a page.

Fix is a toggle function. This means that using Fix without any helping keyword toggles the fixing of the objects on and off.

Szintaxis

Fix [executor list]

Fix On [executor list]

Fix Off [executor list]

Példák

  • To fix executor 1 through 5 on the current page írd be a parancssorba:
[Channel]> Fix On Executor 1 Thru 5

 

  • To toggle executor 3 from changing page, or not to changing page, írd be a parancssorba:
[Channel]> Fix Executor 3
