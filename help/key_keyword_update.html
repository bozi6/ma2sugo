Update Kulcsszó

Az Update kulcsszóhoz, nyomd meg az Update gombot.
Beírhatod az Update -t a parancssorba, vagy röviden Upd.

Leírás

Az update kulcszsó, frissíti a forrásobjektumokat.

Az Update egy függvény kulcsszó.

Szintaxis

Update [Objektum-lista] / [Opció] = [Opció Érték]

Opciók

Az összes elérhető opcióhoz, írd be a parancssorba

[Channel]> Update /?

Az update kulcsszónak az alábbi opciói vannak.

Opció Opció Rövidítés Opció értéke Leírás
addnewcontent an False, True A False ugyanaz mint az originalcontent.
auto not available nincs opció érték Frissíti a preset értékeit az alapvető preset hatókörrel.
cueonly co False, True Megelőzi a trackingot előre.
effects ef False, True Szûri vagy engedélyezi az effekt réteget.
embedded e False, True Beágyazott preseteket hoz létre.
global g nincs opció érték Frissíti a presetet globális értékekkel.
keepactive ka False, True Aktívan hagyja az értékeket.
noconfirm nc nincs opció érték Elnémítja a megerősítő ablakocskát.
originalcontent or False, True Frissíti az eredeti tartalmát egy presetnek, effektnek vagy képnek. A False ugyanaz mint az addnewcontent.
presetfilter p False, True Preset szûrő be/kikapcsolása.
screen not available 1..6 Nézet frissítése a kiválasztott képernyőn.
selective s nincs opció érték Preset frissítése kiválasztott értékekkel.
source so Prog, Output, Dmx In Adatforrás meghatározása.
tracking t False, True Tracking frissítése. A False ugyanaz mint a cueonly.
trackingshield ts nincs opció érték Tracking pajzs használata frissítéskor.
universal u nincs opció érték Preset frissítése univerzális értékekkel.
useselection use Active, Allforselected, Activeforselected, All, Look Kiválasztás beállítása.
values v False, True Sûri vagy engedi az érték réteget.
valuetimes vt False, True Szûri vagy engedi az érték idő réteget.

Példa

1. Frissíti az aktuális képet a kiválasztott executoron, a frissítési alapértelmezésekkel. További információk: Kép frissítése.

[Channel]> Update

2. a 4. kép frissítése, azokkal az értékekkel amik a 4. képből származnak, mint cue only.

[Channel]> Update Cue 4 /originalcontent /cueonly

3. 2. színpreset frissítése, globális értékekkel.az aktív értékek megtartásával a programozóban, és megerősítő ablakocska nélkül. További információ: Presetek frissítése.

[Channel]> Update Preset 4.2 /global /keepactive=true /noconfirm
