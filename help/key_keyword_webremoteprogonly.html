WebRemoteProgOnly Kulcsszó

A WebRemoteProgOnly kulcsszó beírásához, írd be a WebRemoteProgOnly a parancssorba, vagy használd a Wrövidítést.

Leírás

A WebRemoteProgOnly kulcsszó, kiválasztja a web remote lámpák lapján a Prg Onlyopciót (= Csak Programozó).

A WebRemoteProgOnly egy függvény kulcsszó.

A Prg Only funkció kiválasztása a web távoli lámpák oldalán.
