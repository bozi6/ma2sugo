Média Szerver

Média Szerverek olyan eszközök, amik virtuális eszközökre hasonlítanak, de szoftver vezéreltek gyakran dedikált hardware-n.

Média szerverek gyakran masszív mennyiségû attribútumot tudnak felemészteni. Némelyik médiaszerver néhány DMX univerzumot elhasznál. Még az is elõfordul, hogy egy média szerver valamelyik attribútuma 3 DMX csatornát használ a megjelenítéshez (24-bites vezérlés).

Mindegyik médiaszerver különbözik. Valamelyik rugalmasabb a konfigurálásában, valamelyiknek fix készlete van az attribútumokból.

A legtöbb, a videórétegekre van beállítva, ezeket be kell patchelni. Néhánynak mesterrétege van és még kamerarétegek is akadnak.

Más elemei a médiaszervereknek a pixel mapperek, a 3D leképezés/torzítás, hang, effektek, részecske rendszerek, és így tovább.

 

Elég komplex mutatvány, hogy jól használható eszközprofilt készítsünk médiaszerverekhez.

Kiválasztva a Media Server eszköztípust, nem szükséges osztályt(class) hozzárendelni az eszköz profilhoz. Ameddig nincs megjelenítve a médiaszerver az MA 3Dben, addig ok. A videokimenetet a szerverrõl felhasználhatjuk mint CITP adatfolyam az MA 3D-be (errõl bõvebben a CITP használata fejezetben) olvashatsz.
