EffectFade Kulcsszó

To get the EffectFade kulcsszó beírásához, írd be EffectFade vagy rövidenEffectF a parancssorba.

Leírás

EffectFade is a helping keyword to indicate individual effect fade times.

For more information on effect fade times see effects.

Szintaxis

EffectFade

EffectFade [value list]

Examples:

  • To switch the encoder to adjust the effect fade layer írd be a parancssorba:
[Channel]> EffectFade

 

  • To knock out effect fade values of the programmer for the selection or feature írd be a parancssorba:
[Channel]> Off EffectFade

 

  • To set the effect fade time for the selection or feature to 4 seconds írd be a parancssorba:
[Channel]> EffectFade 4

 

  • To set the effect fade for "pan" for the selection to 2 seconds írd be a parancssorba:
[Channel]> Attribute "pan" At EffectFade 2
