Színek választása a csoportopciók közül, hogy minden sorban legyen, vagy össze-vissza.

A következőkben leírjuk hogyan lehet a színeket sorba rendezetten, vagy nem sorba rendezetten beállítani a csoport opcióknál. 

  • All in Order mutatja azokat a lámpákat amik ugyanolyan sorban vannak kiválasztva, mint ahogy a csoportban lettek tárolva. 
  • All Out of Order mutatja a kiválasztott lámpákat, amik eltérő sorrendben vannak kiválasztva, mint ahogy a csoportba lettek mentve.

 A csoportok színeinek megváltoztatásához az All in Order és All Out of Order opciókhoz:

  1. Nyisd meg a  Csoport tárolót.
    - Bökj egy üres részre a képernyőn az. Alapvető ablakok készítése megnyílik.
    - Bökj a Poolsfülre, utána bökj a Groupsgombra. 
  2. Bökj a sárga labdára  a bal felső sarkában a csoport tároló ablakban. 
    A csoport opciók megjelennek. 


    Csoport Opciók
     
  3. Bökj az  All In Order -ra és válassz egy színt a  színtáblázatból vagy a  színkörből. 
    - A választott szín megjelenik mint egy vonal a tároló objektum felirata felett.
  4. bökj  All Out of Order and then choose a color from the color table or the color circle.
    - The color you have chosen is displayed as a line within the pool object right above the font of the pool object.

Let´s say we have selected the color green in All in Order and the color yellow in All Out of Order.

For more information on how to reset and adjust colors, change the pool into sheet style and/or how to use the Hue, Saturation and the Brightness slider see Adjust pool options. 

Fontos:
Some fixtures may have a red line within the pool object right above the font of the pool object. This red line is automatically generated when all fixtures or a part of fixtures are not available due to storing onto another world. 

The pool options are now adjusted to your needs and are saved in your user profile. 

Tipp:
You can also use the Appearance kulcsszó to change the color of one or more pool objects.
