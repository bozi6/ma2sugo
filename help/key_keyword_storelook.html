StoreLook Kulcsszó

A StoreLook kulcsszóhoz, írd be StoreLook a parancssorba, vagy röviden StoreL.

Leírás

A StoreLook eltárolja minden dimmer értékét az összes lámpához a showban.

Ha a dimmer értéke nagyobb mint 0, akkor eltárol minden további attribútumot is.
Ha a dimmer értéke 0, akkor csak a dimmer értékét tárolja el, mert nincs más látható kimenet.

Tipp:
Ha kiakarsz hagyni lámákat vagy csatornákat a StoreLook-ból, deaktiváld a React to Master-t a Patch & Fixtures Schedule-ban.

További információkért, lásd Mik a tárolási opciók?.

StoreLook egy függvény kulcsszó.

Szintaxis

StoreLook [Objektum] / [Opció] = [Opció Érték]

Opciók

A kulcsszó összes opciójának lekérdezéséhez, írd be a parancssorba

[Channel]> StoreLook /?

A storelook kulcsszónak a következõ opciói vannak.

Opció Opció rövidítés Opció értéke Leírás
append a nincs opció érték Új kép hozzáadása.
cueonly co False, True A track forward változásait megelõzi.
effects ef False, True Szûri vagy engedi az effekt réteget.
merge m nincs opció érték Hozzáadja a meglévõ tartalomhoz.
originalcontent or False, True A preset,effekt, vagy kép eredeti tartalmát tárolja. A False ugyanaz mint az addnewcontent.
overwrite o nincs opció érték Felülírja a meglévõ tartalmat.
release rea nincs opció érték Elengedi a tartalmat.
remove r nincs opció érték Eltávolítja a tartalmat.
source so Prog, Output, Dmx In Megadja az adatforrást.
statusmerge sm nincs opció érték Trackállapotot ad a meglévõ képhez.
tracking t False, True Trackingal tárol. A False ugyanaz mint a cueonly.
trackingshield ts nincs opció érték Tracking pajzs használata a storelook során.
values v False, True Szûri vagy engedi az érték réteget.
valuetimes vt False, True Szûri vagy engedi az átúszás és késleltetés réteget.

Példa

Minden dimmer értékének tárolása az összes lámpából, a show-ból, mint 1.kép az 1. executoron.

[Channel]> StoreLook Executor 1
