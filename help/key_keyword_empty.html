Empty Kulcsszó

To get the Empty keyword in the command line, type Empty vagy rövidenEm a parancssorba.

Leírás

With the Empty keyword, you can assign the function empty to executors. Empty means that the executors is deactivated. For more information, see executors - assign a function.

The Empty keyword egy függvény kulcsszó.

Szintaxis

Assign Empty [Executor list]

Példa

To deactivate executor button 101, írd be a parancssorba:

[Channel]> Assign Empty Executor 101

Press Please.

Executor 101 is deactivated. The function empty is assigned.
