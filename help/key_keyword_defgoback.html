DefGoBack Kulcsszó

DefGoBack is a keyword which refers to the selected executor. 

To execute this keyword type DefGoBack  vagy rövidenDefGoB into the command line. 

Szintaxis

DefGoBack

 

DefGoBack calls the previous cue in the selected executor. 

 

Example:

[Channel]> ​DefGoBack 

 

Calls the previous cue in the selected executor. 

 

Tipp:
It is also possible to enter this command by pressing the large Go - button on the console. 

 

For more information on the key Go - see Go - (large).
