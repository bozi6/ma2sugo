IP cím beállítása az onPC-n

Az onPC-n való IP cím megváltoztatásához, először be kell állítani egy IP címet a számítógépeden.

Csak azt az IP címet tudod beállítani, ami az elsődleges Ethernet csatlakozódon van.

Ha az onPC-ről akarsz Art-Net kimenetet, akkor nem szükséges az IP címet az Art-Net tartományból megadni. Az MA onPC automatikusan generál neked egyet.

Ha létrehoztál egy IP címet, az Art-Net tartományban, akkor az onPC nem generál neked ilyen címet, de használja amit beállítottál. Ez lehetséges a második Ethernet kártyán, vagy ugyanazon is.

Az onPC automatikusan fogja használni azt az adaptert Art-Net küldésre, amelyiken Art-Net tartomány van beállítva.

Be kell állítanod egy IP címet a számítógépen, mielőtt elindítod az onPC-t - vagy újra kell indítani a programot, miután megváltoztattad az IP címet a számítógépen.

Ha DHCP-t használsz, akkor győződj meg róla, hogy az IP tartomány ugyanaz legyen az összes MA eszközön, amit csatlakoztatni szeretnél.

Az IP cím beállítása parancssor használatával

Amikor beállítottad az IP címet a számítógépen, akkor használhatod a SetIP parancsot az IP cím megadásához. Fontos, hogy ugyanazt az IP címet írd be, mint amit beállítottál a számítógépen.

Nézd meg a linket a SetIP parancs bővebb ismertetéséhez.

IP cím beállítása a GUI használatával

Kiválaszthatod az IP címet, ha megnyomod a Setup gombot, utána meg a MA Network Control gombot.

Itt kiváaszthatsz egy IP címet amikor az onPC nem csatlakozik munkamenethez. A menüf ejlécén láthatod az aktuális állapotát az onPC-nek. Ha mást látsz mint "Standalone", akkor bökj a Leave Session-ra.

Amikor az onPC standalone-önálló, akkor megnyithatod a legördülő listát a "Station IP:" mellett. Itt kiválaszthatod az IP címet, amit a Windows-ban állítottál be. Emlékezz rá, hogy ezután újra kell indítani az onPC-t, mielőtt használhatnád a kiválasztott IP címet.
