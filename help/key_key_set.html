Set Gomb - Beállítás Gomb

A Set gomb a parancsterületen van az Up gomb alatt.

 

A Set gomb olyan mint az Enter gomb 

Ha egy ablakocska nyitva van, például a Válassz tárolási módot, akkor a Set gomb, funkciója megegyezik azEntergombbal.

  • Megnyomva a Set gombot, elfogadódnak a beállítások a megnyitott ablakocskában.

A Set és MAtricks gombok

  • Megnyomva a Set gombot, aktiválódik az MAtricks beállítások.
[time] : Executing : MAtricks Toggle

Ha az MAtricks gomb zöld, akor az MAtricks engedélyezve van.
 

 

  • Nyomva tartva a Prev + Set gombokat, végrehajtódik az MAtricksInterleave -.
[time] : Executing : MAtricksInterleave -
  • Nyomva tartva a Next + Set gombokat, végrehajtódik az MAtricksInterleave +.
[time] : Executing : MAtricksInterleave +
  • Nyomva tartva a  + Prev + Set gombokat, végrehajtódik az MAtricksBlocks -.
[time] : Executing : MAtricksBlocks -
  • Nyomva tartva az  + Next + Set gombokat, végrehajtódik az MAtricksBlocks +.
[time] : Executing : MAtricksBlocks +

További információk, lásd: MAtricks.

A Set gomb a képernyőkonzol dőlésszögének állításához

Ha kézzel akaord állítani a képernyőkonzolt, ami kiengedi a konzol motorját.

  • Nyomva tartva az  + Set.
