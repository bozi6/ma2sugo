Képek átszámozása

A kép számai változhatnak. Ez nem használható a képek átrendezésére és asorrendjük megváltoztatására.

Amikoregy képet szerkesztesz a Szekvencia Executor Lapon, egy ilyesmi ablakocska jön elő:

Jelek sora választható ki az újraszámozáshoz, de ne feledd, hogy az új számok nem haladhatják meg a kiválasztás előtti és utáni számokat.

Az előugró ablak két legfelső beviteli mezője használható tartomány beállítására, vagy egyszerűen csak egy tartományra van.

Ez alatt található az újraszámozás szakasz. Itt van az "New Number-Új szám" beállításának lehetősége. Ez lesz az első kép száma, ha kép tartományt választasz.

Megadható egy "lépésszélesség". Ez akkor releváns, ha jelzési tartományt választanak. Ezt használjuk a lépésméret beállításához a céltartomány között.

Ha a beviteli mezők megfelelően vannak kitöltve, a Renumber-Újraszámozás gombra bökéssel elvégezheted az újraszámozást - ez bezárja az előugró ablakot is.

Egyéb funkciók

Az alul található Egyéb funkciók részben, négy gomb található.

A "Szín módosítása" leírás lejjebb.

"Block Cue(s) - A kép(ek) blokkolása", és a "Unblock cue(s) - Kép(ek) blokkolásának feloldása" a követéssel kapcsolatos. Ezt a Mi a Követés fejezet írja le.

A "Delete Cue(s) - Kép(ek) törlése" törli a kijelölt képeket. Ezzel a Képek törlése témakör foglalkozik.

Változtasd meg a kép színét

A kép színt rendelhet a Szekvencia Executor lap sorához.

Ez a parancssor használatával vagy a Change Color - Szín módosításagombra bökéssel érhető el az kép szám szerkesztése ablakocskában.

A Megjelenés kulcsszó erre a parancssoron keresztül történik.

Hacsak nem a parancssor bevitelével adja meg a színt, megjelenik a Szín kiválasztása előugró ablak.

A lapbeállítások vezérlik, hogy ez a szín látható-e, és hogyan lehet a cellák háttérszíne. További információ erről a Lapbeállítások - Megjelenítés - Cue színoszlopok oldalon található.

"Block Cue(s)", "Unblock Cue(s)" are related to tracking. This is described in the What is Tracking topic.

"Delete Cue(s)" will delete the selected cues. This is discussed in the Delete Cues topic.

Change cue color

A cue can have a color assigned to the row in the Sequence Executor Sheet.

This can be done using the command line or by tapping the Change Colorbutton in the Edit Cue Numbers pop-up.

The Appearance keywordis used to do this via the command line.

Unless you use the command line input to specify a color then the Select Color pop-upappears.

The sheet options control if this color is visible and how may cells will have the background color. Read more about this in the Sheet Options - Display - Cue Color Columns.
