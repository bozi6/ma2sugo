Prev (Previous) Gomb - Elõzõ Gomb

A Prev gomb, a parancsterületen van, balra a Set gombtól.

 

Ha egy ablakocska megnyílik, például a Válassz Tárolási módot ablakocska, akkor a fókuszt áthelyezheted a következõ opciókra a Prev megnyomásával.

  • Megnyomva a Prev gombot, végrehajtódik a Previous kulcsszó. A lámpa/csatorna a legmagasabb ID-vel kiválasztódik.

  • Nyomvatartva az  + Prev gombokat, végrehajtódik a PrevRow kulcsszó. További információkért:MAtricks.
[time] : Executing : PrevRow
  • Megnyomva a Prev + Next gombokat, végrehajtódik az All kulcsszó.
