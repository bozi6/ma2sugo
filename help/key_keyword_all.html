All Kulcsszó

Az All kulcsszó beírásához, írd be az All -t a parancssorba.

Leírás

Az All egy funkció ami az MAtricks Single X tulajdonságát none (0)-ra állítja, ami azt jelenti, hogy újraválasztja az összes lámpát, (vagy újraválasztja minden lámpa oszlopait, ha az Interleave aktív).

További információ: MAtricks.
