ListUserVar Kulcsszó

To get the ListUserVar keyword in the command line, type ListUserVar vagy rövidenListUs a parancssorba.

Leírás

The ListUserVar keyword egy függvény kulcsszó to get the user specific variables and their values in the Command Line Feedback window displayed.

Szintaxis

ListUserVar [Filter]

Példák

  • To get all user specific variables in the command line feedback window displayed, írd be a parancssorba:
[Channel]> ListUserVar

 

  • To get all user specific variables beginning with f in the command line feedback window displayed, írd be a parancssorba:
[Channel]> ListUserVar f*
