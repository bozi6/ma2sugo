EffectHZ Kulcsszó

To get the EffectHZ kulcsszó beírásához, írd be EffectHZ vagy rövidenEffectH a parancssorba.

Leírás

EffectHZ is a helping keyword to indicate individual effect speed in Hertz.

For more information see Effects.

Szintaxis

EffectHZ

EffectHZ [value list]

Példák

  • To switch the encoder to the effect speed layer írd be a parancssorba:
[Channel]> EffectHZ

 

  • To knock out any effect speed values of programmer for the selection or feature írd be a parancssorba:
[Channel]> Off EffectHZ

 

  • To set the effect speed for the selection or feature to 4 Hertz írd be a parancssorba:
[Channel]> EffectHZ 4

 

  • To set the effect speed for "pan" for the selection to 2 Hertz írd be a parancssorba:
[Channel]> Attribute "pan" At EffectHZ 2
