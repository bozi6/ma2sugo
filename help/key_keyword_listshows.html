ListShows Kulcsszó

To get the ListShows keyword in the command line, type ListShows vagy rövidenListS a parancssorba.

Leírás

The ListShows keyword egy függvény kulcsszó to display the show files on the selected drive in the Command Line Feedback window.

Szintaxis

ListShows [Filter]

Példák

  • To get all show files in the command line feedback window displayed, írd be a parancssorba:
[Channel]> ListShows

 

  • To get all show files beginning with f in the command line feedback window displayed, írd be a parancssorba:
[Channel]> ListShows f*
